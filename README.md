# Voicey
Voicey is a project for modulating and demodulating three audio signals by the following formula
```bash
s(t) = x1(t)cos(ω1*t)+ x2(t)cos(ω2*t)+ x3(t)sin(ω2*t)
```
to simulate sending and receiving signals through channels.
